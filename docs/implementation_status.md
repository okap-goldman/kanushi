# 実装状況

## 機能間の依存関係
### 認証・ユーザー管理依存
- タイムライン機能 → ユーザー管理機能（投稿者情報、認証）
- フォロー機能 → ユーザー管理機能（フォロワー/フォロイー情報）
- イベント機能 → ユーザー管理機能（主催者/参加者情報）
- ショップ機能 → ユーザー管理機能（出品者/購入者情報）

### 外部サービス依存
- 動画投稿 → YouTube API連携
- 画像投稿 → Box連携
- 音声投稿 → PodBean API連携
- 通知機能 → Firebase Cloud Messaging

### 決済依存
- イベント参加 → Stripe決済
- ショップ購入 → Stripe決済

### その他の依存関係
- イベント関連投稿 → イベント機能、投稿機能
- ハイライト/コメント → 投稿機能
- 発見機能のおすすめ投稿 → 投稿機能、ユーザー分析
- AIチャット → DeepSeek R1 API連携
